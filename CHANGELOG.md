# Changelog - Advanced Data Analyzer Pro

## v2.0.0 - 2024-01-XX

### 🎉 Nuevas Características
- **IA Gratuita**: Integración con Google Gemini 2.5 Flash
- **ML Avanzado**: 3 algoritmos (Random Forest, XGBoost, LightGBM)
- **Base de Datos**: SQLite integrado con historial
- **Exportación**: Reportes PDF profesionales
- **UI Moderna**: CustomTkinter con tema oscuro

### 🔧 Mejoras Técnicas
- Arquitectura MVC modular
- Asistente IA unificado (Gemini + OpenAI)
- Feature engineering automático
- Sistema de backup automático
- Validación cruzada avanzada

### 🐛 Correcciones
- Error de encoding Unicode en Windows
- Problemas de threading con matplotlib
- Serialización JSON en asistente IA
- Manejo robusto de columnas categóricas

### 📦 Dependencias Actualizadas
- pandas >= 2.0.0
- customtkinter >= 5.2.0
- google-generativeai >= 0.8.0
- xgboost >= 2.0.0
- lightgbm >= 4.0.0

## v1.0.0 - 2024-01-XX

### Características Iniciales
- Análisis básico de datos Excel
